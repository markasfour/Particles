About
Particles is created using the open source Processing programming language and is based off of the Liquid Particles Google Chrome Experiment. This implementation of Particles provides a simpler, less chaotic, and longer lasting visual experience of the particle movement along with features of pushing particles away and adding particles at will. The Processing source code for this project can be found at Github.
